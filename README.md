# Kodaal-Guppy

**Kodaal Guppy** is a local-first prompt workspace — capture prompts from browser, CLI, and IDE surfaces, search and reuse them on your machine, with no cloud required.

Full pitch, quick start, and run guide: [`kodaal-guppy/README.md`](kodaal-guppy/README.md).
